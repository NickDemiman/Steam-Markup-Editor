# В оригинале

1. ctrl+z не отменяет выделение тегами
2. Комбинации привязаны к раскладке

Пример:

```plain
[h1]Заголовок 1[/h1]
[h2]Заголовок 2[/h2]
[h3]Заголовок 3[/h3]
[b]Жирный текст[/b]
[u]Подчеркивание[/u]
[i]Курсив[/i]
[strike]Перечеркнутый[/strike]
[spoiler]Спойлер![/spoiler] Пробел [spoiler]Еще раз спойлер[/spoiler]
[noparse]
[h1]Заголовок 1[/h1]
[h2]Заголовок 2[/h2]
[h3]Заголовок 3[/h3]
[b]Жирный текст[/b]
[u]Подчеркивание[/u]
[i]Курсив[/i]
[strike]Перечеркнутый[/strike]
[/noparse]
[hr]
[code]
example:
server:
    address: consul.ext
    port: 6379
[/code]
```
